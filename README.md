# Training Plan e-commerce site

A static, JAMstack e-commerce site I created for cycling coach and YouTuber, Dylan Johnson, while I was working for Specialized BC as a technical writer, DevEx engineer and API integrations specialist.

For more information, check out the [project's page on my portfolio](https://chrisgb.dev/projects/training-plan-store).

I originally created the site back in 2022, and haven't updated it since. This repo is archived for professional development purposes.

Note, Specialized shuttered Today's Plan and Zone5Cloud, so any links/api calls to the Today's Plan API or UI won't work.
